# YSS FS Frontend

基于 Vue 3 + TypeScript + Vite + Ant Design Vue 的文件存储微应用骨架。

## 技术栈

- Vue 3
- TypeScript
- Vite
- Vue Router
- Pinia
- Ant Design Vue
- qiankun 兼容

## 启动

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
pnpm build:jsp
pnpm build:standalone
```
